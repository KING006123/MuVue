# Vue 2 考前速记（基于你的上机内容）

## 一、Vue 实例三件套

```js
let vm = new Vue({
    el: "#app",       // 挂载点：Vue管哪个div
    data: { ... },    // 数据
    methods: { ... }  // 方法
})
```

- data里的数据在模板中用 `{{变量名}}` 显示
- methods里的方法用 `this.变量名` 访问data数据
- `this` 指向当前Vue实例

## 二、指令大全（背！考试必考）

| 指令 | 简写 | 作用 | 例子 |
|------|------|------|------|
| `{{ }}` | - | 插值表达式，显示数据 | `{{xm}}的简历` |
| `v-text` | - | 纯文本输出，替换全部内容 | `v-text="myinfo.name"` |
| `v-html` | - | 解析HTML标签输出 | `v-html="myinfo.school2"` |
| `v-bind` | `:` | 单向绑定属性 | `:src="imgurl"` |
| `v-on` | `@` | 绑定事件 | `@click="changepic"` |
| `v-if / v-else` | - | 条件渲染（真删假建） | `v-if="!ifshow"` |
| `v-show` | - | 显示/隐藏（CSS display） | `v-show="ifshow2"` |
| `v-for` | - | 循环渲染 | `v-for="item in list"` |
| `v-model` | - | 双向数据绑定 | `v-model="inputvalue"` |

### 关键区别常考：
- **v-if vs v-show**：v-if真删DOM元素，v-show只是CSS隐藏。频繁切换用v-show
- **v-text vs v-html**：v-text不解析HTML标签，v-html解析
- **v-bind vs v-model**：v-bind数据→页面（单向），v-model页面↔数据（双向）

## 三、v-for 三种遍历方式

```html
<!-- 1. 遍历数组：item是值 -->
<li v-for="item in school.colleges">{{item}}</li>

<!-- 2. 带索引：(item, index) -->
<li v-for="(item,index) in school.colleges">{{index+1}}---{{item}}</li>

<!-- 3. 遍历对象：(value, key, index) -->
<li v-for="(value,key,index) in myinfo">{{key}}---{{value}}</li>
```

---

## 四、Class 绑定四种方式（上机3）

```html
<!-- 1. 数组常量 -->
<div v-bind:class="['box','box3']"></div>

<!-- 2. 数组变量 -->
<div v-bind:class="[classa,classb]"></div>

<!-- 3. 布尔对象：true就加这个class -->
<div v-bind:class="{'box':ifshow, 'box3':ifshow}"></div>

<!-- 4. 混合 -->
<div v-bind:class="['box', {'box3':!ifshow}]"></div>
```

## 五、生命周期钩子（上机7，记住执行顺序）

```
beforeCreate → created → beforeMount → mounted 
                  ↓                       ↓
            data初始化完成          页面渲染完成
                  ↓                       ↓
            beforeUpdate ← → updated（数据变化时触发）
```

每个钩子干啥：
- **beforeCreate**：实例刚创建，data/methods还不能用
- **created**：data/methods可用了，页面还没渲染（常用：请求数据）
- **beforeMount**：模板编译好了，还没挂载到页面
- **mounted**：页面渲染完成，可以操作DOM（常用：初始化）
- **beforeUpdate**：数据变了，页面还没重新渲染
- **updated**：页面重新渲染完成

## 六、组件（上机8）

**全局组件**（任何Vue实例都能用）：
```js
Vue.component("组件名", { template: "<p>内容</p>" })
```

**局部组件**（只在当前Vue实例中用）：
```js
let vm = new Vue({
    components: {
        "组件名": 组件对象
    }
})
```

**组件本质**：一个带template的JS对象

## 八、购物车核心逻辑（理解）

```
勾选 → changechecked() → 计算总价gettotalprice()
全选 → changeallchecked() → 所有item.ischecked=true → 算总价
加号 → add(index) → num++ → 重新算总价
减号 → sub(index) → num-- → 重新算总价
```

注意购物车用了 **mounted()** 初始化总价，页面一加载就调 `gettotalprice()`
